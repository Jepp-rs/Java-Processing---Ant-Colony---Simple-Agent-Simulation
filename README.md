# Ant Colony Simulation

Este es un pequeño proyecto hecho en **Processing (Java)** como práctica personal para aprender programación orientada a objetos y simulaciones simples.

La simulación muestra una colonia de hormigas que se mueven por el mapa buscando comida. Cuando una hormiga detecta comida cercana, se dirige hacia ella, la recoge y vuelve a la colonia. Cada vez que la colonia recibe comida, aumenta el contador de suministros y eventualmente se generan más hormigas.

El objetivo del proyecto no es ser una simulación biológica realista, sino practicar conceptos de programación y experimentar con comportamientos básicos de agentes.

## Controles

Click del mouse: crea una nueva fuente de comida en la posición del cursor.

## Características actuales

- Movimiento aleatorio de las hormigas
- Detección de comida por distancia
- Recolección y transporte de comida
- Regreso automático a la colonia
- Generación de nuevas hormigas basada en suministros recolectados
- Creación de comida mediante el mouse

## Cosas que todavía no tiene

Por ahora el proyecto es bastante simple. No incluye:

- Feromonas
- Comunicación entre hormigas
- Optimización de caminos
- Otros comportamientos más complejos

## Cómo ejecutar

1. Instalar **Processing**
2. Abrir el archivo `.pde`
3. Ejecutar el sketch

## Nota

Este proyecto fue hecho principalmente para aprender y practicar programación.
